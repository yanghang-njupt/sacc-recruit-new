### Git 使用

```javascript
把远程仓库拉下来
git clone [url]
创建自己的分支
git branch [name]-dev(统一命名name-dev)
切换分支
git checkout [branch-name]

先创建本地仓库
git init
上传代码到本地仓库
git add .
合并冲突
git pull
上传提示信息
git commit -m '[info]'
上传代码
git push [name]-dev
```
